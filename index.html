Setup required

Connect Firebase for live voting

This app uses Firebase Realtime Database so your group can vote live and see results update in real time. Setup takes about 5 minutes and is completely free.

Create a Firebase project

Go to console.firebase.google.com → Add project → give it a name (e.g. "dark-mofo-2026") → Continue until done.

Enable Realtime Database

In your project: Build → Realtime Database → Create database → Start in test mode → Enable.

Get your config

Project Settings (gear icon) → Your apps → Web app (</>) → Register app → copy the firebaseConfig object.

Paste your config below

Paste the full firebaseConfig = { ... } object here, then save the HTML file.

Deploy to Vercel (optional)

Once your config is pasted and saved: go to vercel.com/new → drag and drop this HTML file → Deploy. You'll get a shareable URL instantly.
